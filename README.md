# huge
[![Build Status](https://travis-ci.org/davidosomething/huge.png?branch=master)](https://travis-ci.org/davidosomething/huge)

Console canvas line and flood-fill program.

## Getting Started

### Run tests
```
npm install
grunt
```

### Run program
```
./bin/huge.js
```

### Use in JavaScript
```javascript
var huge = require('huge');
```
